# StarQuiz Game
Aplicação desenvolvida em ReactJS & Redux.

----------

## Dependências
- Node.js + NPM
<br>Versão: **node -v** 
<br>Versão: **npm  -v**
- Adicionar dependências (node_modules)
```bash
  npm install
```   

----------

## Executar Aplicação
```bash
  npm start
```
