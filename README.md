# StarQuiz Game
Aplicação desenvolvida em ReactJS & Redux.

----------

## Requisitos
- Node.js + NPM
- Instalar dependências (node_modules)
```bash
  npm install
```   

----------

## Executar Aplicação
```bash
  npm start
```
